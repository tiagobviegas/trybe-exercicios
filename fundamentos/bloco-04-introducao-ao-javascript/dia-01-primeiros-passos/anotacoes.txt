VARIÁVEIS

let movie = "Avengers"; // string literal
let score = 10.89; // number literal
let isValid = true // boolean
let name; // undefined
let color = null; // redefinição

Variáveis do tipo "let" permitem redeclaração e reatribuição de valores

Fazendo referência a script dentro do HTML usamos a tag <script>, atribuindo o direcionamento para o arquivo js que será lido para implementação do cógio na página.

	<script src="script.js"></script>

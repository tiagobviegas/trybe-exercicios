Entendendo sobre o prevent default

A função preventDefault() impede que o comportamento padrão do elemento. No caso abaixo, o botão Enviar é um elemento "button" de type submit que, por padrão, atualiza a página após o evento de "click" e retorna os elementos ao estado original.

Por isso, a informação inserida no input e que deveria aparecer na tag p abaixo é perdida e o nome enviado não aparece como innerHTML do parágrafo.

Então, adicionamos ao elemento button, que recebe a ação de click, a função preventDefault() e assim impedimos que ele atualize a página. Assim, a informação do input é inserida na tag p pelo innerHTML.
